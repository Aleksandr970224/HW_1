1) Посмотреть где я - pwd
2) Создать папку  -  mkdir qa_30_free 
3) Зайти в папку  - cd qa_30_free
4) Создать 3 папки  - mkdir group_1 group_2 group_3 
5) Зайти в любоую папку - cd group_1
6) Создать 5 файлов (3 txt, 2 json) - touch qq1.txt qq2.txt qq3.txt qq4.json qq5.json
7) Создать 3 папки mkdir qq_6 qq_7 qq_8 
8. Вывести список содержимого папки - ls 
9) + Открыть любой txt файл - cat qq1.txt
10) + написать туда что-нибудь, любой текст. - cat >>qq1.txt
11) + сохранить и выйти. - cntrl + c 
12) Выйти из папки на уровень выше- cd ..
—
13) переместить любые 2 файла, которые вы создали, в любую другую папку.
  - mv group_1/qq1.txt group_2/qq1.txt ; mv group_1/qq2.txt group_2/qq2.txt
14) скопировать любые 2 файла, которые вы создали, в любую другую папку. - 
 cp group_1/qq3.txt group_2/qq3.txt ; cp group_1/qq4.json group_2/qq4.json
15) Найти файл по имени - find -name qq1.txt 
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает. -  grep hi qq1.txt
17) вывести несколько первых строк из текстового файла - head -4 qq1.txt 
18) вывести несколько последних строк из текстового файла - tail -3 qq1.txt 
19) просмотреть содержимое длинного файла (команда less) изучите как она работает. - less qq1.txt 
20) вывести дату и время - date
